11.6.20
There are infinitely many primes.
Suppose there is a finite number of primes \(\small{p_1, p_2, p_3, ..., p_n}\). Let \(\small{P}\) be an integer defined as \(\small{P = p_1 p_2 p_3 ... p_n + 1}\). Let \(\small{p}\) be a prime dividing \(\small{P}\). Then \(\small{p}\) cannot be any prime \(\small{p_1, p_2, p_3, ..., p_n}\) becuase \(\small{p}\) would divide \(\small{P - p_1 p_2 p_3 ... p_n = 1}\) which is impossible. So \(\small{p}\) must be another prime.
